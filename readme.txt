== Description ==
The "Votecount for Balatarin" plugin shows the number of votes on "Balatarin.com" your posts get and allows users to vote it themselves.

Features:

* Display the number of Balatarin votes for each post/page
* Configure the style of the badge
* Disable the button on Pages

More info:

== Installation ==

1. Unzip the `votecount-for-balatarin.zip` file. 
2. Upload `votecount-for-balatarin` folder to your `wp-contents/plugins/` folder.
3. In your WordPress administration, go to the Plugins page
4. Activate the "Votecount for Balatarin" plugin and a subpage will appear in your Options menu. You can configure the appearance there.
5. For each post/page you want the votes to be shown, create a custom field in the write panel of that post/page with the Name of "bvcurl" and Value of [for example] `2010/7/28/2135212` (if the post/page url in Balatarin is `http://balatarin.com/permlink/2010/7/28/2135212`.

If you find any bugs or have any ideas, please [email us](tafreevar.weblog@gmail.com).
